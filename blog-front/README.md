# blog-front

`blog-front` 是博客项目的前端工程，使用 React、TypeScript 和 Vite 构建。

## 技术栈

- React 18
- TypeScript
- Vite
- Ant Design
- Zustand

## 本地运行

```bash
npm install
npm run dev
```

开发环境下会把 `/api` 请求代理到 `http://localhost:8080`。

## 构建命令

```bash
npm run build
```
